These scripts each labelled as an exam, creates a DOCX file for an individualized exams and emails it to them. 

EXAM_1 - combinational exam
- Question 1 creates a boolean function that needs to be solved for
- Question 2 has a function that needs to be minimized (kmaps)

EXAM_2 - combinational arithmetic exam
- Question 1 is to create a function
- Question 2 is to create a comparator

EXAM_3 - sequential exam
- Question 1 generates a bit detection sequence that needs to be solved with an FSM
- Question 2 a problem in Verilog that uses Q1

EXAM_4 - c code
- C code that needs to be Converted to Verilog - One key part of this is to use on-chip memory IP
